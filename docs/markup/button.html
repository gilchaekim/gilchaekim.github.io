Button Group

// 최근, 자주, 내계좌 선택 버튼
+button_group({
  btns:[
    '최근',
    '자주',
    '내계좌',
  ]
})
// 금액 입력 버튼 그룹
+button_group({
  btns:[
    '+1만',
    '+5만',
    '+10만',
    '전액',
  ]
})


Attribute - Button

Attributes
Property Description Type Default
label 버튼 텍스트 string -
size 버튼 사이즈 : large, medium, small string medium
isIcon 아이콘 포함 여부 Boolean false

Attribute - Button Group

Attributes
Property Description Type Default
btns 버튼 그룹 텍스트 리스트 array -
className 커스텀 클래스 string -
